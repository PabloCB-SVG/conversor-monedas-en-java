# conversor-monedas-en-java
Conversor de Monedas en Java Este proyecto permite realizar conversiones de distintas monedas en tiempo real utilizando la API de ExchangeRate. El programa está desarrollado en Java, con un menú interactivo en consola que facilita al usuario elegir moneda de origen, así como destino. Incluye manejo de excepciones y consumo de datos en formato JSON.
